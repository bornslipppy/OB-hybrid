### BLOCKER-8 (Role-collapse data audit) — PARTIALLY RESOLVED
Data pull (2026-05-25): Multi-seat proxy from cross-dashboard user roles (N=8,381). 79.2% of role-covered accounts have 2+ users at steady state. BUT SMB role coverage is only 30.4% — absence of role records ≠ single-seat for SMB. Single-user accounts cluster at median 3 listings ($194 MRR), below the 5–20 listing Wizard target. Data does not directly measure first-30-days onboarding-moment logins. **Decision:** Role-collapse premise reframed as an onboarding-moment claim, not a steady-state claim. The owner-operator is the primary setup actor during the pre-Call-1 window. Decision rule updated: opinionated UX survives if Q4.1 delegation rate in V1 ≤50% (replaces original ≥75% single-user threshold, which was pre-data). §1.1 Premise 3 updated.

### BLOCKER-6 (KPI baseline refresh) — PARTIALLY RESOLVED
Data pull (2026-05-25): SMB on-time graduation rate 30.6% (was 18% in prior brief — updated in SM-12 baseline); median SMB onboarding duration 31 days vs. 20-day SLA; active onboarding backlog overdue rate 69.9%; post-graduation churn within 60 days 59.5% of churned cohort (n=79), median 51 days graduation-to-churn. Key strategic insight from data: graduation appears to be a process milestone, not an activation milestone — customers can graduate with shallow adoption and churn within 60–90 days. This reinforces the Wizard's core bet (real account state at Call 1 → durable activation). SM-10 and SM-11 baselines not yet pulled — those sub-items remain open.
